# Tableau_Food_Cleaned_Data
**Food Delivery Analytics Dashboard | Tableau**
----> **Overview**

**Link: https://public.tableau.com/app/profile/girish.bhosale8411/viz/Food_Cleaned_Data/Food_Cleaned_Data**

This project was developed as part of a Tableau Data Visualization Challenge using a food delivery dataset. The goal was to transform raw order data into meaningful business insights through interactive visualizations and dashboards.

**Key Visualizations**
----> Delivery Outcome

Analyzes city-wise delivery performance, delivery status distribution, and Average Order Value (AOV).

----> Revenue vs Customer Experience

Examines monthly revenue trends, customer ratings, and new vs returning customer behavior across cities.

----> Restaurant Performance

Evaluates restaurant performance using dynamic metrics such as Average Rating, Total Revenue, and Total Orders.
